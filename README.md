# beschetnova-msk

Сайт для юриста Ирины Бесчётновой.

## Структура проекта
- `index.html` — главная страница с квизом и быстрой формой
- `services/family-law.html` — семейное право
- `services/court-protection.html` — судебная защита
- `services/pretrial-settlement.html` — досудебное урегулирование
- `services/real-estate.html` — сопровождение сделок с недвижимостью
- `services/bankruptcy.html` — сопровождение процедур банкротства
- `assets/css/styles.css` — стили сайта
- `assets/js/main.js` — логика квиза и подготовки сообщений для мессенджеров
- `assets/images/` — изображения проекта

## Что уже подключено
- многостраничная структура под SEO и рекламу
- квиз на главной странице
- быстрые формы на главной и на страницах услуг
- подготовка готового сообщения для отправки в Telegram и WhatsApp без backend

## Локальный запуск
Сайт статический, поэтому достаточно открыть `index.html` в браузере.

Для корректной проверки можно поднять простой локальный сервер:

```bash
python3 -m http.server 8080
```

После этого откройте:

```text
http://localhost:8080
```
